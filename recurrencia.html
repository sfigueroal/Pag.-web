¿Qué es la Recurrencia?

La recurrencia es un concepto utilizado en matemáticas y programación para definir una solución en términos de versiones más pequeñas del mismo problema. Es la base de muchas soluciones recursivas.

Un ejemplo clásico es el cálculo de la serie de Fibonacci:

int fibonacci(int n) {
  if (n == 0) return 0;
  if (n == 1) return 1;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

Árboles en Estructura de Datos

Un árbol es una estructura jerárquica formada por nodos. Cada nodo tiene un valor y puede tener hijos (otros nodos). El nodo superior se llama raíz.

Un tipo común es el árbol binario, donde cada nodo tiene como máximo dos hijos: izquierdo y derecho.

Los recorridos de un árbol binario son:

  • Inorden (izquierda, nodo, derecha)
  • Preorden (nodo, izquierda, derecha)
  • Postorden (izquierda, derecha, nodo)
// Recorrido Inorden en Java
void inorden(Node nodo) {
  if (nodo != null) {
    inorden(nodo.izq);
    System.out.print(nodo.valor + " ");
    inorden(nodo.der);
  }
}

Ejercicios Propuestos con Solución

1. Factorial de un número

Enunciado: Crear una función recursiva que calcule el factorial de un número entero positivo.

int factorial(int n) {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
}

2. Recorrido Preorden de un árbol

Enunciado: Escribir una función que recorra un árbol binario en preorden.

void preorden(Node nodo) {
  if (nodo != null) {
    System.out.print(nodo.valor + " ");
    preorden(nodo.izq);
    preorden(nodo.der);
  }
}

3. Búsqueda de un valor en un árbol binario de búsqueda

Enunciado: Crear una función recursiva que verifique si un valor existe en un árbol binario de búsqueda (BST).

boolean buscar(Node nodo, int valor) {
  if (nodo == null) return false;
  if (nodo.valor == valor) return true;
  if (valor < nodo.valor) return buscar(nodo.izq, valor);
  return buscar(nodo.der, valor);
}
