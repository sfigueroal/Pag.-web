¿Qué son los Fundamentos de Programación?

Son los conceptos básicos que necesitas para comenzar a programar. Incluyen el uso de variables, operadores, estructuras de control (condicionales y bucles), funciones y estructuras de datos simples.

Variables y Tipos de Datos

Las variables permiten almacenar datos en memoria. Por ejemplo, en Java:

int edad = 20;
float altura = 1.60f;
String nombre = "Suemy";

Operadores

Sirven para realizar operaciones matemáticas o lógicas:

  • +: suma
  • -: resta
  • *: multiplicación
  • /: división
  • &&: y lógico
  • ||: o lógico

Estructuras de Control

Permiten controlar el flujo del programa:

// Condicional
if (edad >= 18) {
  System.out.println("Eres mayor de edad");
} else {
  System.out.println("Eres menor de edad");
}

// Bucle
for (int i = 0; i < 5; i++) {
  System.out.println("Iteración: " + i);
}

Funciones

Son bloques de código reutilizables:

public static int sumar(int a, int b) {
  return a + b;
}

Ejercicios Propuestos

  1. Declara una variable para cada tipo de dato y muestra su valor por consola.
  2. Escribe un programa que verifique si un número es par o impar.
  3. Implementa una función que calcule el factorial de un número.

Soluciones

1. Declaración de variables:

int numero = 5;
float precio = 12.5f;
char letra = 'A';
boolean activo = true;
String texto = "Hola";
System.out.println(numero + ", " + precio + ", " + letra + ", " + activo + ", " + texto);

2. Par o impar:

int num = 7;
if (num % 2 == 0) {
  System.out.println("Es par");
} else {
  System.out.println("Es impar");
}

3. Factorial:

public static int factorial(int n) {
  int resultado = 1;
  for (int i = 1; i <= n; i++) {
    resultado *= i;
  }
  return resultado;
}
